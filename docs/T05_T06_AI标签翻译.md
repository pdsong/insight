# T05 & T06 — AI 自动标签、关键词提取、中文翻译与摘要

## 完成时间
2026-02-26

## 做了什么

### T05: AI 自动标签与关键词提取

#### `Insight.AI.Tagger` 模块
- 设计了分类 Prompt，从 25 个系统标签中选择 1-3 个匹配标签
- 提取 3-5 个核心关键词存入 `keywords` 字段
- 自动关联标签到 `news_tags` 表

#### 验证结果
输入: `"Show HN: Open-source AI code editor"`
输出:
```
tags: ["AI", "开源", "编程"]
keywords: "AI代码编辑器,开源项目,GitHub,开发者工具,人工智能"
```

### T06: AI 中文翻译与摘要生成

#### `Insight.AI.Summarizer` 模块
- 翻译英文标题为中文 → `title_zh` 字段
- 生成 50-100 字中文摘要 → `summary_zh` 字段
- 保留技术术语英文原文（如 LLM、API 等通用缩写）

#### 验证结果
输入: `"Show HN: Open-source AI code editor"`
输出:
```
title_zh: "Show HN：开源 AI 代码编辑器"
summary_zh: "该项目在 GitHub 上开源，是一款集成 AI 功能的代码编辑器..."
```

### 集成到爬虫流程
`Scraper.Worker` 流程现在为：**抓取 → 入库 → AI 标签 → AI 翻译**

### Mix 任务
- `mix insight.tag` — 手动对未标注新闻执行 AI 标签
- `mix insight.translate` — 手动对未翻译新闻执行 AI 翻译
