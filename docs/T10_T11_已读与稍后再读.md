# T10 + T11 — 已读状态标记 & 稍后再读

## 完成时间
2026-02-26

## 做了什么

### T10 — 已读状态标记
| 功能 | 说明 |
|------|------|
| 标记已读 | 点击 👁 图标手动标记单条新闻为已读 |
| 一键全部已读 | 页面顶部"全部标为已读"按钮，批量标记当前页所有新闻 |
| 视觉区分 | 已读新闻卡片 opacity-50 半透明显示 |
| 图标切换 | 未读 `hero-eye` → 已读 `hero-eye-slash` |

### T11 — 稍后再读（收藏）
| 功能 | 说明 |
|------|------|
| 收藏/取消收藏 | 点击🔖图标 toggle 收藏状态 |
| 图标反馈 | 未收藏 `hero-bookmark`（空心）→ 已收藏 `hero-bookmark-solid`（实心，黄色） |
| 状态持久化 | 基于 `user_interactions` 表，`action=bookmark` |

### 共享基础设施
- `refresh_interactions/3`：高效重载指定新闻的交互状态
- `has_action?/3`：统一判断 like/dislike/bookmark/read 状态
- `get_user_id_from_assigns/1`：模板中安全获取当前用户 ID
