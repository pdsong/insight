# T09 — 用户自定义标签管理

## 完成时间
2026-02-26

## 做了什么

### 1. 创建 `TagLive.Index` LiveView
- **系统标签区**：只读展示，badge 列表，附带说明"由 AI 自动标注"
- **用户标签区**：
  - 创建：输入框 + 按钮
  - 编辑：点击铅笔图标 → 内联编辑模式
  - 删除：点击垃圾桶图标，带确认对话框
- **权限保护**：系统标签不可修改/删除

### 2. 路由
- `/tags` 放在 `require_authenticated_user` live_session 中
- 未登录用户访问时自动跳转到登录页

### 3. 导航
- 已登录用户在 root layout 顶部菜单显示"标签管理"链接
