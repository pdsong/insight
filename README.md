## 📖 Introduction

**Insight** is more than just a HackerNews client. It is an intelligent reading platform built with **Elixir** and **Phoenix LiveView** that learns from your reading habits. By tracking your interactions (likes, bookmarks, reads) and leveraging LLM capabilities, Insight continuously refines your personal interest profile to recommend the content that matters most to you—while intentionally injecting "Serendipity" (破圈) recommendations to keep you discovering new horizons.

## ✨ Features

- ⚡️ **Real-time LiveView UI**: Ultra-fast, SPA-like experience without writing complex JavaScript.
- 🕷 **Built-in HN Web Scraper**: Automated data synchronization pulling the latest and hottest HackerNews items.
- 🧠 **Dynamic Interest Profiles**: Automatically calculates tag weights based on your reading interactions (`Like` | `Bookmark` | `Read` | `Dislike`).
- 🤖 **AI-Powered Recommendations**: 
  - **Personalized Reasons**: Context-aware, one-sentence recommendations generated by AI based on your top interest tags.
  - **Serendipity (破圈)**: Randomly surfaces out-of-bubble, high-quality content to broaden your horizons.
- 📊 **Insight Radar & Daily Summaries**: Visualizes your learning paths and provides AI-summarized tech digests.
- 🔖 **Bookmarks & Timelines**: A beautifully crafted timeline to review your footprints and favorite articles.
- 🛡 **Noise Cancellation**: Advanced user-defined blocking management (mute by keywords or tags).

## 🛠 Tech Stack

- **Backend**: [Elixir](https://elixir-lang.org/), [Phoenix Framework](https://www.phoenixframework.org/)
- **Frontend**: [Phoenix LiveView](https://hexdocs.pm/phoenix_live_view), HTML5, [TailwindCSS](https://tailwindcss.com/) + [DaisyUI](https://daisyui.com/)
- **Database**: [SQLite](https://sqlite.org/) (via `exqlite` for effortless local deployments)
- **AI Integration**: Custom LLM Integrations (Prompts & Recommender Agents)



## 🚀 Getting Started

To start your Insight server locally, follow these steps:

### Prerequisites

- **Elixir** (v1.17 or later)
- **Erlang/OTP** 
- **Node.js** (optional, for some asset compilation if extended)

### Setup & Run

1. **Clone the repository:**
   ```bash
   git clone https://github.com/pdsong/insight.git
   cd insight
   ```

2. **Install dependencies:**
   ```bash
   mix deps.get
   ```

3. **Setup the SQLite database:**
   ```bash
   # 1. Create the database (if starting fresh)
   mix ecto.create
   # 2. Run migrations (create table structures)
   mix ecto.migrate
   # 3. Run seed data (insert default system tags)
   mix run priv/repo/seeds.exs
   ```

4. **Fetch initial HackerNews data:**
   ```bash
   mix insight.crawl
   ```

5. **Start the Phoenix endpoint:**
   ```bash
   mix phx.server
   ```
   *(Or run inside IEx: `iex -S mix phx.server`)*

Now you can visit [`localhost:4000`](http://localhost:4000) from your browser.
